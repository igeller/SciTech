1. copy the index.html in the protected folder into the folder/ repository area where you want a Password
2. show that file in finder, and then open it in web browswer
3. type in the password that you want to use for your site
4. it will say wrong password, however copy the numbers and letters that appear after the # in the site name
   and create a folder in the same folder where the index.html file appears with that string of numbers after
   the #
5. add the content that you want to be password protected to the newly created folder with the long number and
   letter string
